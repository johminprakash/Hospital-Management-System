&#x09;				||**Hospital Management System**||

&#x09;				==============================



A comprehensive management solution designed to digitize and streamline hospital workflows, patient records, and staff coordination.



**Overview**

This project provides a robust platform for healthcare facilities to manage daily operations. It reduces paperwork, minimizes scheduling conflicts, and ensures that patient data is easily accessible to authorized personnel.



**Key Features**

*Patient Management*: Easy registration and medical history tracking.

*Doctor Dashboard*: View assigned patients and manage consultation hours.

*Appointment System*: Automated scheduling with conflict detection.

*Billing \& Invoicing*: Integrated billing for consultations and room charges.



**Tech Stack**

*Language*: Python 3.x

*Database*: SQLite / PostgreSQL

*Frontend*: HTML5, CSS3, JavaScript



**Installation \& Setup**



*1. Clone the Repository*

&#x20;  git clone https://github.com/johminprakash/Hospital-Management-System

&#x20;  cd Hospital-Management-System



*2. Set Up Virtual Environment* 	

&#x20;  python -m venv venv

&#x20;  Windows: venv\\Scripts\\activate | Mac: source venv/bin/activate



*3. Install Dependencies*

&#x20;  pip install -r requirements.txt



*4. Run the Application*

&#x20;  python run.py
